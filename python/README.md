### Реалізація на Python 

**Опис**
Об'єктно-орієнтована та імперативна реалізація ігрового ШІ (алгоритм MinMax із Alpha-Beta відсіканням) та механіки гри "k-в-ряд". Стан гри інкапсульовано всередині класу `ConnectK`, де ігрове поле представлено як змінний двовимірний масив (список списків). На відміну від декларативних парадигм, алгоритм покладається на явні цикли, пряму мутацію стану та створення глибоких копій об'єкта для обходу дерева рішень.
Головна функція: `alpha_beta(game, depth, alpha, beta, player)`

**Особливості**
* **Інкапсуляція стану:** Уся логіка (перевірка перемоги, генерація ходів, евристика) та дані поля зберігаються в об'єкті класу `ConnectK`.
* **Імперативні конструкції:** Використання класичних циклів `for` та `while` для обходу матриці та перевірки напрямків.
* **Мутабельність та копіювання:** Симуляція ходів для ШІ реалізована через створення незалежних копій гри за допомогою `copy.deepcopy(game)` з подальшою мутацією скопійованої дошки.
* **Степенева евристика:** Оцінка ліній базується на швидкому підрахунку через степені десятки (наприклад, `10 ** (p1 - 1)`), що робить код лаконічним.
* **Ручна рекурсія:** Рекурсивний обхід дерева Мінімакс прописано вручну, з явним збереженням локальних максимумів/мінімумів.
