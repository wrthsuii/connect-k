#  Гра "k в ряд" (Connect K)

> k в ряд "падають "розмір колодязю: n x m". Параметри: n>=5, m>=6, k=[3..5] - кількість в ряд для виграшу. +MinMax (Alpha–Beta). Визначаються заборонені комірки.

## Структура проєкту

```
connect-k/
├── haskell/
│   ├── game.hs          # Реалізація ігрової логіки та ШІ (MinMax) на Haskell
│   └── README.md        # Опис Haskell реалізації
├── python/
│   ├── game.py          # ООП-реалізація ігрової логіки та ШІ на Python
│   └── README.md        # Опис Python реалізації
├── web/
│   ├── index.html       # Головна сторінка фронтенду (UI)
│   ├── script.js        # Логіка фронтенду (взаємодія з бекендом, відмальовка)
│   └── style.css        # Стилі інтерфейсу
├── game.pl              # Основна логіка гри та ШІ на Prolog
├── server.py            # Flask-сервер (API), що з'єднує фронтенд із Prolog-логікою
├── COMPARISON.md        # Детальне порівняння трьох парадигм (Prolog, Python, Haskell)
└── README.md            # Загальний опис проекту та інструкції із запуску
```
